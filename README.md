 # AI/ML Engineering Internship Tasks
This repository contains my completed tasks for the AI/ML Engineering Internship at DevelopersHub Corporation**.
The goal of these tasks is to build practical skills in data analysis, machine learning, and basic AI applications.

## Tasks Completed

### Task 1: Iris Dataset Visualization

* Loaded and explored the Iris dataset using Pandas
* Performed data analysis using `.info()` and `.describe()`
* Created visualizations:

  * Scatter plots
  * Histograms
  * Box plots
* Tools: Pandas, Matplotlib, Seaborn

### Task 2: Stock Price Prediction

* Retrieved historical stock data using `yfinance`
* Used features like Open, High, Low, Volume
* Trained a **Linear Regression model**
* Compared actual vs predicted closing prices
* Tools: yfinance, Scikit-learn, Matplotlib

### Task 3: Heart Disease Prediction

* Cleaned and analyzed medical dataset
* Trained a **Logistic Regression model**
* Evaluated using:

  * Accuracy
  * Confusion Matrix
  * ROC-AUC Score
* Applied feature scaling to improve performance
* Tools: Pandas, Scikit-learn

### Task 4: General Health Chatbot

* Built a chatbot using a transformer model (DistilGPT2)
* Applied **prompt engineering** for safe responses
* Implemented basic **safety filters**
* Tools: Hugging Face Transformers

### Task 5: Mental Health Support Chatbot

* Developed an empathetic chatbot using pre-trained model
* Designed prompts for supportive and gentle responses
* Added safety handling for sensitive inputs
* Tools: Transformers (DistilGPT2)
